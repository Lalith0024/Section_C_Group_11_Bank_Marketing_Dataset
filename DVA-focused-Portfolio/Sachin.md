## Portfolio Projects

### 1. Audio Profiling & Viral Track Characteristics

Repository: [The-Entertainment-Media-Project](https://github.com/jaiswalsachin49/The-Entertainment-Media-Project)

This data analytics project focuses on the entertainment and media domain to understand what makes a song go viral. The project includes a clear, step-by-step workflow covering data extraction, cleaning, exploratory data analysis (EDA), and statistical analysis.

The case study covers:

- Problem statement and stakeholder context
- Data extraction and cleaning workflow
- Exploratory data analysis (EDA) and statistical analysis
- Tableau dashboard for visualization
- Key insights on viral track characteristics

### 2. EV Adoption Trends & Grid Impact Matrix

Repository: [EV-Adoption-Trends-Grid-Impact-Matrix](https://github.com/jaiswalsachin49/EV-Adoption-Trends-Grid-Impact-Matrix)

This project validates an end-to-end data analysis pipeline to study the impact of Electric Vehicles (EVs) on the power grid. It is optimized for a data analyst portfolio, logically structuring the workflow from data to final reporting.

The case study covers:

- Problem statement and stakeholder context
- Data extraction and cleaning
- Exploratory data analysis (EDA) and statistical analysis
- Final reporting and dashboarding using Tableau
- Key insights and recommendations

### 3. Bank Marketing Data Analysis

Repository: [Section_C_Group_11_Bank_Marketing_Dataset](https://github.com/Lalith0024/Section_C_Group_11_Bank_Marketing_Dataset)

This banking analytics project studies a Portuguese banking institution's direct marketing campaign data to understand which customer groups are more likely to subscribe to a term deposit.

The case study covers:

- Problem statement and stakeholder context
- Dataset source and scope (UCI Portuguese Bank Dataset)
- Cleaning and transformation summary
- Propensity model
- KPI framework
- Tableau dashboard visualization
- Key insights and recommendations for optimizing term-deposit conversion
