# Jagruthi Pulumati - DVA-Focused Portfolio

Live Portfolio: [https://dva-portfolio-swart.vercel.app/](https://dva-portfolio-swart.vercel.app/)

This is my Data Visualization & Analytics portfolio, built as a simple static website using HTML and CSS. It presents my capstone work as portfolio-ready case studies with problem framing, dataset scope, cleaning and transformation work, KPI frameworks, dashboard screenshots, insights, and recommendations.

## Portfolio Projects

### 1. Impact of COVID-19 on Maternal & Newborn Health

Repository: [SectionC_Group6_Pregnant_Women_COVID19](https://github.com/Jag2007/SectionC_Group6_Pregnant_Women_COVID19)

This healthcare analytics project studies maternal mental health and neonatal outcomes during the COVID-19 pandemic using survey-based data from the Pregnancy during the COVID-19 Pandemic study conducted in Canada.

The case study covers:

- Problem statement and stakeholder context
- Dataset source and scope
- Data cleaning and transformation summary
- KPI framework
- Dashboard screenshot and links
- Key insights
- Recommendations and expected impact

Key metrics:

- 5,389 cleaned records
- 3,415 g average birth weight
- 19.4% preterm birth rate
- 9.78% NICU admission rate
- 29.36% C-section rate
- 51.54 / 100 average pandemic stress score

### 2. Bank Marketing Campaign Analysis

Repository: [Section_C_Group_11_Bank_Marketing_Dataset](https://github.com/Lalith0024/Section_C_Group_11_Bank_Marketing_Dataset)

This banking analytics project studies a Portuguese banking institution's direct marketing campaign data to understand which customer groups are more likely to subscribe to a term deposit.

The case study covers:

- Problem statement and stakeholder context
- Dataset source and scope
- Cleaning and transformation summary
- KPI framework
- Tableau dashboard screenshots and link
- 3-5 key insights
- Recommendations and expected impact

Key metrics:

- 41,188 raw rows
- 41,176 cleaned rows
- 27 cleaned columns
- 11.27% overall conversion rate
- 14.74% cellular conversion rate
- 65.11% prior success conversion rate
