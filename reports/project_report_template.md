# Project Report Template

Use this outline to draft the final written report before exporting it as `project_report.pdf`.

## 1. Cover Page

- Project title
- Sector
- Team ID and team members
- Faculty mentor
- Institute
- Submission date

## 2. Executive Summary

- Problem
- Approach
- Key insights
- Key recommendations

## 3. Sector and Business Context

- Sector overview
- Decision-maker / stakeholder
- Why this problem matters

## 4. Problem Statement and Objectives

- Formal problem definition
- Scope
- Success criteria

## 5. Data Description

- Source citation and access link
- Dataset size and coverage
- Key columns
- Data quality issues

## 6. Cleaning and Transformation

- Major cleaning steps
- Assumptions made
- Output dataset description

## 7. KPI Framework

- KPI definitions
- Formulae
- Why each KPI matters

## 8. Exploratory Analysis

- Major trends
- Segment-level insights
- Visual summaries

## 9. Statistical Analysis

- Method used
- Results
- Business interpretation

## 10. Dashboard Walkthrough

- Dashboard objective
- Executive view
- Operational view
- Filters and interactivity

## 11. Key Insights

- 8-12 decision-oriented findings

## 12. Recommendations

- 3-5 recommendations
- Expected business impact

## 13. Limitations and Next Steps

- Data limitations
- Method limitations
- Suggested future work
